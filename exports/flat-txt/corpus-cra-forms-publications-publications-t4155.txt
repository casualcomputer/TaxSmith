# T4155 Old Age Security Return of Income (OASRI) Guide for Non-Residents

This pamphlet is useful to non-residents of Canada receiving Old Age Security (OAS) payments. It provides detailed information on completing Form T1136, ''Old Age Security Return of Income''.
